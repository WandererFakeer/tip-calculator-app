## Overview

This solution provides the Per Person Tip Amount and Per Person Total Amount in their own local Currency.

User can enter their total Bill Amount in the `Bill` input. This input accepts numbers only .

Then, they can choose any tip %. It offers a `Custom` button, clicking which will open up an input field, where they can enter whatever % amount they want to enter.

Finally, they can enter they number of people, who will share this bill. They can enter correct number in `Number of people` input field.
If user enters "0" as `Number of people`, they will see an alert, saying `Can't be zero`. So, they need to enter correct number for `Number of people` input field.

After user completes their input entry, it will show the Per Person Tip Amount and Per Person Total Amount, as `Tip Amount / person` and `Total / person` respectively.

The user has an option to reset everything with `Reset` button. This button will be disabled at the start, and will only be enabled if the user provides all 3 data in their respective fields.

This is a screen-size friendly calculator.
